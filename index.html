PSV Hybrid Propulsion

Repositório de previsão de demanda energética para propulsão híbrida (exemplo educacional).

Visão geral

Este projeto contém um preprocessador, um modelo LSTM e scripts para gerar dados sintéticos e treinar/testar modelos.

  • Dados: data/energy_demand.csv
  • Scripts principais: generate_synthetic_data.py, app.py, main.py
  • Código do modelo: src/prediction/lstm_model.py

Como executar localmente

Exemplo rápido para Windows PowerShell:

python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py

Ou execute scripts específicos:

python generate_synthetic_data.py
python main.py
