PSV Hybrid Propulsion

Demo interativa de previsão de demanda energética com upload de CSV e backend de treino/predição.

1. Upload de dados

Selecione um arquivo CSV com colunas timestamp e power_demand. O gráfico será exibido automaticamente.

2. Treinar modelo

Use este botão para treinar o modelo no backend com os dados carregados ou com data/energy_demand.csv.
